# Coding Style Quick Ref
## Use C style conventions 
- Constants: `CONSTANT`
- Variables: `camelCase`
- Functions: `camelCase`
- Import names:
  - `import cv2 as cv`
  - `import numpy as np`
- Use: Main and Function structure for easy debug
# RPI and Arduino Interfacing
- SCL: A5 to Pin 5
- SDA: A4 to Pin 3
- GND: GND to Pin 6
